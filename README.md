# TigerPlate 🐯

Live nutrition tracker for Clemson University dining halls. Pulls real-time menu data from all campus dining locations with full macro breakdowns, AI meal identification, and an AI nutrition coach.

## Features
- **Live Menu Data** — Pulls today's actual menu from Schilletter, Douthit Hills, and McAlister
- **AI Meal Scanner** — Snap a photo, Gemini identifies it and pulls nutrition
- **AI Coach (Tiger)** — Chat with an AI that knows exactly what's being served right now
- **Macro Tracking** — Log meals and track calories, protein, carbs, fat
- **Station Grouping** — Browse by dining station (Clean Eats, The Grill, Pizza, etc.)

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** FastAPI (Python)
- **AI:** Google Gemini 2.5 Flash
- **Data:** Live GraphQL from Clemson MyDiningHub
